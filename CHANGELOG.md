## [1.1.0] - 2026-07-12

### Added
- Complete Unreal Engine 5 presentation project for the Alder Staff asset
- Substance 3D Painter texturing workflow
- Game-ready low-poly asset export
- High-poly to low-poly normal map baking workflow
- Git LFS configuration for large project assets
- Expanded repository organization for Unreal Engine integration

### Changed
- Refined overall silhouette of the wooden staff shaft
- Improved edge flow and topology throughout the model
- Created a complete UV unwrap for the low-poly mesh
- Applied stylized PBR materials for the wood, crystal, and metal accents
- Imported and configured the finished asset inside Unreal Engine 5
- Established presentation lighting for portfolio-quality renders
- Improved project organization and ignored generated Unreal Engine files to reduce repository bloat

### Notes
This refinement milestone transforms the original classroom assignment into a complete game-ready asset workflow. The project now demonstrates the full environment art pipeline including mesh refinement, UV unwrapping, high-to-low baking, physically based texturing in Substance 3D Painter, and presentation inside Unreal Engine 5. Additional polish, cinematics, and portfolio presentation will be completed during later Capstone milestones.
