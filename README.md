# Stylized Fantasy Staff

> **SNHU Game Programming Capstone – Artifact 1**

## Overview

This repository contains the source files and development history for my Stylized Fantasy Staff, originally created for **GRA 202 – 3D Modeling and Animation** and later refined as part of my **GAM 495 Game Programming Capstone** at Southern New Hampshire University.

The goal of this capstone refinement is to transform an early coursework project into a polished portfolio piece that better reflects my current skills in 3D asset creation and game development.

---

## Capstone Refinement Goals

During the capstone, this project will be refined through several iterations, including:

- Improved topology
- Proper UV unwrapping
- High-quality PBR texturing
- Updated materials
- Enhanced presentation renders
- Turntable animation
- Improved documentation
- Unreal Engine presentation (if applicable)
